# Gerçek Orthopedics

Welcome to the **Gerçek Orthopedics** project! This repository contains the source code for a modern and user-friendly website for an orthopedic solutions center based in Ankara, Turkey. The website provides information about services, announcements, blogs, and more.

## 🌟 Features

- **Responsive Design**: Fully optimized for all devices.
- **Multi-language Support**: Available in Turkish and English.
- **Dynamic Content**: Blog posts, announcements, and service details.
- **Admin Dashboard**: Manage announcements, blogs, and other content.
- **SEO Optimized**: Ensures better visibility on search engines.

## 🚀 Technologies Used

- **Framework**: [Next.js](https://nextjs.org/) (React-based framework)
- **Styling**: Tailwind CSS
- **State Management**: Context API
- **Database**: PostgreSQL (via Prisma ORM)
- **Deployment**: Vercel

## 📂 Project Structure

```
├── app/                # Application pages and routing
├── components/         # Reusable UI components
├── hooks/              # Custom React hooks
├── lib/                # Utility functions and configurations
├── public/             # Static assets
├── scripts/            # Database and setup scripts
├── styles/             # Global styles
├── types/              # TypeScript type definitions
```

## 🛠️ Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/gercekorthopedics.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd gercekorthopedics
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Set up environment variables**:
   - Create a `.env` file in the root directory.
   - Add the required environment variables (refer to `.env.example` if available).

5. **Run the development server**:
   ```bash
   npm run dev
   ```

6. **Open the app**:
   Visit [http://localhost:3000](http://localhost:3000) in your browser.
